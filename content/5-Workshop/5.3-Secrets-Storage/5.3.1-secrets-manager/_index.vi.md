### Secrets Manager

**Console:** tìm "Secrets Manager" → **Store a new secret**

1. Secret type: **Other type of secret**
2. Tab **Plaintext**, dán JSON cấu hình ứng dụng (MONGO_URI, JWT, MoMo, SES…)
3. Secret name: `webfood/prod/app-secrets` → **Store**
4. 📌 Ghi lại **Secret ARN**
