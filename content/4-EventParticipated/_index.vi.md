Trong thời gian thực tập tại chương trình **First Cloud AI Journey (FCAJ)** tại AWS Việt Nam, tôi đã tham gia **02 sự kiện cộng đồng** tại văn phòng AWS (Tầng 26, Tòa nhà Bitexco). Mỗi buổi giúp mở rộng kiến thức Cloud/AI, soft skills và kết nối với cộng đồng học viên.

### [Event 1 — Community Day AWS (30/05/2026)](4.1-Event1/)

**Tên sự kiện:** Community Day AWS — First Cloud Journey AI  

**Thời gian:** 30/05/2026  

**Địa điểm:** Tầng 26 - Văn phòng AWS Việt Nam (Tòa nhà Bitexco, Số 2 Hải Triều, Phường Sài Gòn, TP.HCM)  

**Vai trò:** Người tham dự  

**Nội dung chính:** Các phiên chia sẻ về AWS Cloud Quest & Floci, demo hackathon **SynthHunter**, soft skills (confidence, procrastination), sản phẩm cộng đồng (Tử vi Đại Việt) và tư duy DevOps (*DevOps Before Disaster*).  

**Giá trị đạt được:** Nâng nhận thức về học AWS có lộ trình, cost awareness khi thực hành trên account thật, cách trình bày dự án AI/hackathon, và các kỹ năng mềm hỗ trợ quá trình thực tập / triển khai **WebFood**.

→ [Xem bài thu hoạch chi tiết Event 1](4.1-Event1/)

### [Event 2 — Meetup First Cloud Journey AI (06/06/2026)](4.2-Event2/)

**Tên sự kiện:** Meetup First Cloud Journey AI  

**Thời gian:** 06/06/2026  

**Địa điểm:** Tầng 26 - Văn phòng AWS Việt Nam (Tòa nhà Bitexco, Số 2 Hải Triều, Phường Sài Gòn, TP.HCM)  

**Vai trò:** Người tham dự  

**Nội dung chính:** Sáu bài trình bày gồm Docker/containerization, AWS WAF + Machine Learning (NIDS), multiplayer realtime với Godot + AWS WebSockets, GraphRAG trên Amazon Neptune & Bedrock, teamwork hiệu quả, và lộ trình nghề nghiệp Helpdesk → Sysadmin → Cloud/DevOps.  

**Giá trị đạt được:** Liên hệ trực tiếp với kiến trúc **WebFood** (WAF, WebSocket, serverless), mở rộng góc nhìn về container, bảo mật, GraphRAG và lộ trình nghề nghiệp Cloud.

→ [Xem bài thu hoạch chi tiết Event 2](4.2-Event2/)
