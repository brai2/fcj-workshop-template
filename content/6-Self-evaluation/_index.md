During my internship at **Amazon Web Services Viet Nam Company Limited** from **17/04/2026** to **30/07/2026**, under the **Workforce Bootcamp – First Cloud AI Journey** program, I had the opportunity to learn AWS cloud fundamentals and apply them in a real project.

I designed and deployed **WebFood** — an online food-ordering platform on an **AWS Serverless** architecture (CloudFront, WAF, S3, API Gateway, Lambda, EventBridge, SQS, DynamoDB, Secrets Manager, SES, CloudWatch). Through this work I improved full-stack development skills (React, Node.js/Express), cloud architecture design, console-based (click-ops) deployment, workshop documentation, and technical reporting.

Regarding work ethic, I aimed to complete weekly worklog tasks on time, followed program processes, and proactively asked mentors when I faced difficulties configuring AWS services.

### Strengths

* Understood the end-to-end serverless flow of WebFood: from CloudFront/WAF to Lambda, EventBridge, SQS, and WebSocket.
* Manually deployed resources on the AWS Console in the correct dependency order (Secrets Manager, S3, DynamoDB, IAM, Lambda, API Gateway, CDN).
* Delivered Proposal, Workshop, and Worklog documentation tied to a real project instead of only following sample labs.
* Proactively learned additional services (EventBridge rules, CloudWatch Alarms, SNS) and added screenshots to the report.

### Needs Improvement

* Strengthen **IaC** skills (AWS SAM / CloudFormation) to reduce reliance on click-ops and make environments easier to recreate.
* Improve **communication and presentation** when explaining complex architecture (event-driven, WebSocket) to listeners less familiar with cloud.
* Build stronger **operations mindset**: monitor Free Tier cost, tear down resources regularly, and handle cold starts and DLQ failures more systematically.
* Practice **CI/CD** and automated testing for frontend/backend before deploying to AWS.

### Post-internship Direction

After the FCAJ program, I plan to deepen my knowledge of AWS services (especially serverless and observability), evolve WebFood closer to production (Cognito, CI/CD pipeline, cost optimization), and build experience toward a Cloud / Backend Developer role.
